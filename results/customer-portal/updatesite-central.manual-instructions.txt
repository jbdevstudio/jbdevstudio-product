The easiest way to install this update is to select 'Help > Check for Updates' from within a running devstudio instance. Doing so will provide you with a list of all the available updates found at ${devstudio.update.site}

To use this zip for offline update (or if blocked by firewall or proxy when doing searching for updates from the sites above):

1. Download, but DO NOT unpack, the zip

2. Launch Devstudio or Eclipse ${eclipse.version}

3. Select Help > Install new software

4. Click Add and browse for the zip

5. Uncheck 'Group items by category' to reveal the features on the site

6. Select the feature(s) you wish to install and accept the licenses

7. Restart when prompted
